# Vinitha R - Personal Portfolio

A minimalist, modern personal portfolio for Vinitha R built with React. Clean design inspired by contemporary developer portfolios with a focus on typography, spacing, and smooth interactions.

## 🎨 Design Features

- **Minimalist Aesthetic**: Clean, professional layout with white, light gray, and black color palette
- **Smooth Animations**: Subtle fade-in effects and smooth scroll behavior
- **Responsive Design**: Fully responsive for desktop and mobile devices
- **Modern Typography**: Uses Poppins and Inter fonts for a contemporary feel
- **Soft Shadows & Rounded Corners**: Subtle depth with refined visual hierarchy

## 📂 Sections

1. **Hero** - Introduction with name, subtitle, and social links
2. **About** - Professional background and interests
3. **Experience** - Work experience and roles
4. **Projects** - Featured projects with descriptions
5. **Skills** - Technical and soft skills
6. **Achievements** - Awards and recognitions
7. **Education** - Academic background
8. **Contact** - Contact form and email
9. **Footer** - Social links and copyright

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd vinitha-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The portfolio will open at `http://localhost:3000`

## 🛠️ Build

To create a production build:

```bash
npm run build
```

This creates an optimized build in the `build` folder ready for deployment.

## 📱 Responsive Design

The portfolio is fully responsive and works seamlessly on:
- Desktop (1920px and above)
- Tablet (768px - 1024px)
- Mobile (320px - 767px)

## 🎯 Customization

### Update Personal Information

Edit the following files to customize with your information:

- `src/components/Hero.js` - Name, subtitle, and social links
- `src/components/About.js` - About section content
- `src/components/Experience.js` - Work experience
- `src/components/Projects.js` - Projects
- `src/components/Skills.js` - Skills
- `src/components/Achievements.js` - Achievements
- `src/components/Education.js` - Education
- `src/components/Contact.js` - Contact information

### Styling

All styles are modular and located in corresponding `.css` files. Modify colors, spacing, and animations in:
- `src/index.css` - Global styles
- `src/components/*.css` - Component-specific styles

## 🚀 Deployment

### Deploy to Netlify

1. Build the project:
```bash
npm run build
```

2. Connect your repository to Netlify or drag and drop the `build` folder

### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel
```

## 📦 Dependencies

- **React** - UI library
- **React DOM** - React rendering
- **Lucide React** - Icon library
